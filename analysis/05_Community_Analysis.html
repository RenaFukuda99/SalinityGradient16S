05_Between_Sample_Diversity

Rena Fukuda

Goals

  1. Load in phyloseq data with rooted true.
  2. Evaluate sequencing depth and remove a sample.
  3. Normalize the read counts between samples.
  4. Calculate community disimilarity indices
  1. Sorenson: Shared species as a binary value (abundance-unweighted)
  2. Bray-Curtis: Shared abundant species (abundance-weighted)
  3. weighted UniFrac: Consider abundant species and branch lenghts
  1. Visualize the commmunity data with two unconstrained ordinations.
  1. PCoA: Linear method, eigenvalue: how much variation is explained by each axis
  2. NMDS: Multiple dimensions into 2 or 3
  1. Run statistics with PERMANOVA and betadispR.

Normalize read counts

# Function to scale reads: http://deneflab.github.io/MicrobeMiseq/ 
# Scales reads by 
# 1) taking proportions
# 2) multiplying by a given library size of n
# 3) rounding 
# Default for n is the minimum sample size in your library
# Default for round is floor

matround <- function(x){trunc(x+0.5)}

scale_reads <- function(physeq, n = min(sample_sums(physeq)), round = "round") {
  
  # transform counts to n
  physeq.scale <- transform_sample_counts(physeq, function(x) {(n * x/sum(x))})
  
  # Pick the rounding functions
  if (round == "floor"){
    otu_table(physeq.scale) <- floor(otu_table(physeq.scale))
  } else if (round == "round"){
    otu_table(physeq.scale) <- round(otu_table(physeq.scale))
  } else if (round == "matround"){
    otu_table(physeq.scale) <- matround(otu_table(physeq.scale))
  }
  
  # Prune taxa and return new phyloseq object
  physeq.scale <- prune_taxa(taxa_sums(physeq.scale) > 0, physeq.scale)
  return(physeq.scale)
}
